📘 정보보안개론 필기 핵심

+ 1·2차 모의필기 오답 역추적

  • 틀린 문항 위주로 개념 정리
  • 암호·네트워크 보안·접근통제·시스템 보안 우선
📝 정보보안기사 기출 반복
「수제비 정보보안기사」 또는 이기적 시리즈
+ 정처기 기출 앱 (무료)
  • 이론서보다 기출 5개년 반복이 핵심
  • 모르는 개념 나올 때만 이론서 참고
  • 이동시간 앱 풀이 → 귀가 후 오답노트
💻 CS 지식 면접 필수
OS → KOCW 반효경 교수 강의 (무료 명강의)
DB → 유튜브 "코딩애플" DB 강의
네트워크 → 유튜브 "널널한 개발자"
  • 프로세스·스레드, 메모리관리, TCP/IP, 정규화 우선
  • 면접 단골 개념 위주로 압축 정리
🛡️ 보안 솔루션 면접 단골
별도 교재 없이 블로그·유튜브 검색
방화벽 / IDS·IPS / WAF / SIEM / DLP / EDR
  • 종류 → 역할 → 동작 원리 → 한계점 4단계로 정리
  • "이 솔루션 설명해봐" 유형 면접 대비
📰 보안 동향 매일 루틴
boannews.com / AhnLab TIP

  • 주 1~2개 이슈 → 노션에 3줄 요약 스크랩
  • "최근 보안 이슈 말해봐" 면접 질문 대비
🎯 정처기 병행
「수제비 정보처리기사」 필기·실기 시리즈
